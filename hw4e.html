Homework #4: Switch Statements

Part 4: Choosing Lists Formats

Click to Show/Hide Instructions

Create a webpage that contains a script that uses a form text box for one of three different list formats. Use jQuery to show/hide instructions and validate input data. Use the following to determine which format is desired. Your script should use a switch statement to help determine this.

1 - unordered list

2 - ordered number list

3 - ordered letter list

After determining which list format has been chosen, write an appropriate header line and then use a for statement to write 3 lines in the chosen list format.

For Example, if the inputted value is 1, then write:

Unordered List

  • List item 1
  • List item 2
  • List item 3

For Example, if the inputted value is 2, then write:

Ordered Number List

  1. List item 1
  2. List item 2
  3. List item 3

For Example, if the inputted value is 3, then write:

Ordered Letter List

  1. List item 1
  2. List item 2
  3. List item 3
Finally if the inputted value is NOT 1, 2, or 3 then display an error message using jQuery.


Type in a number from 1 to 3 to generate a list. Read the instructions for further information about the type of list you will generate.
